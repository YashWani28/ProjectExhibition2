Welcome to Sorting Visualizer
With this sorting Visualizer, you can visualize the sorting of an array as it happens. You can choose the type of sorting algorithm you want to visualize. You can also adjust the speed and the number of elements in the array using the 2 sliders provided. We have color coded each step of the sorting to make your experience great. Take a look at the color code given at the right
Color Code
  • YELLOW - comparing 2 elements
  • BLUE - origninal color of all elements
  • GREEN - if 2 elements are in the right order
  • RED - if 2 elements are in the wrong order
  • PURPLE - elements that have been sorted
